üì¶ Push an image to your local Zot registry

Zot docs: zotregistry.dev v2.1.8
Local registry: http://localhost:5001/ 

# Build your image
docker build -t myapp:latest .

# Tag for local Zot (port 5001 maps to internal 5000)
docker tag myapp:latest localhost:5001/myorg/myapp:latest

# Push to Zot
docker push localhost:5001/myorg/myapp:latest

# Pull it back (optional)
docker pull localhost:5001/myorg/myapp:latest

Your Kind nodes are already configured to mirror localhost:5001.

üåê Example Ingress (nginx + *.localhost)

Use ingressClassName: nginx and a host under *.localhost with TLS:

apiVersion: networking.k8s.io/v1
kind: Ingress
metadata:
  name: myapp
  namespace: default
  annotations:
    cert-manager.io/cluster-issuer: "quick-kind-ca"
spec:
  ingressClassName: nginx
  rules:
  - host: app.localhost
    http:
      paths:
      - path: /
        pathType: Prefix
        backend:
          service:
            name: myapp
            port:
              number: 80
  tls:
  - hosts:
    - app.localhost
    secretName: app-localhost-tls

TLS is automatically signed by your local quick-kind-ca ClusterIssuer.
